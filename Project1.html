WEB SCANNING APPLICATION

The application's function is to allow users on the local area network to scan the network.

Technologies included: Linux/Centos 7, Apache Web Server, PHP, Cronjob, Nmap.

TThis application is hosted locally on a Centos 7 OS. The Apache server will deploy the website on the local area network. Cronjob is used to update the PHP script every 10 minutes, updating the web application data. Nmap will scan the local area network within the script.

Application goal: This application will allow users with proper permission to scan the LAN and get information such as IP address, port information, mac address, and device vendors.

Switch to root user 
Su -
					note: Use root password to login to root account.
Install Httpd, Php, Nmap packages 
yum install httpd php nmap
Create a network.php and nmap.html file in the html directory 
 cd /var/www/html
 
 touch network.php nmap.html
Create Cronjob running Nmap And Outputing to results to file 
 crontab -e

Note: you will be in text editor press "I" to go into insert mode

*/10 * * * * nmap 192.168.1.0/24 > /var/www/html/nmap
Input Php script to network.php file 
<?php

echo "Server Timestamp: ";
echo date("h:i:sa");
					
echo "<pre>";
include("nmap.html");
echo "</pre>";
					
?>
